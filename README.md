# Jumpscare Plugin
**Inspired by the [Isaac mod by joemkdd](https://steamcommunity.com/sharedfiles/filedetails/?id=3410284721)**

Plays a jumpscare at random.

You can select from a list of default images/sounds.

Selecting multiple will randomly pick between all of the selections.

Preview images/sounds by clicking the files name.

Uploading custom assets for use is also possible (image format must be **gif/png/jpg**, sound format must be **wav/mp3**, file size **less than 30mb**)

**Default values:**
- Timer between 10-10000 seconds
- Timer display off
- Withered Foxy jumpscare

## Installation
- Enter **/xlplugins** in the chat window or select **Dalamud Plugins** in the System menu.
- In the **All Plugins** tab, search for **Jumpscare** by Piggy Guy and install it.
